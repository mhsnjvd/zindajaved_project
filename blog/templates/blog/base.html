Adventures in probability

One more toss

April 16, 2017

Suppose Asma and Babar decide to play a coin tossing game. We suppose that Asma tosses $n$ fair coins while Babar tosses $n+1$ fair coins. The player with more heads wins the game. What is the probability that Babar wins the game?

Let us define the following random variables.

$$A_n = \text{# of heads thrown by Asma in $n$ tosses},$$ $$B_n = \text{# of heads thrown by Babar in $n$ tosses}.$$
We are trying to find the probability $P(B_{n+1} > A_n)$. Let us now condition on the first $n$ tosses. If $B_{n+1} > A_n$, then either $B_n = A_n$, which means that the final toss by Babar decides the game, or $B_n > A_n$, in which case Babar has already won the game in the first $n$ tosses. Mathematically, we can write this using the law of total probability $$P(B_{n+1} > A_n) = P(B_{n+1} > A_n | B_n > A_n) P(B_n > A_n)+ P(B_{n+1} > A_n | B_n = A_n)P(B_n = A_n).$$

We have $$P(B_{n+1} > A_n | B_n > A_n) = 1.$$ However, $$P(B_{n+1} > A_n | B_n=A_n ) = 1/2,$$ since Babar's $(n+1)^{st}$ toss is a head only with probability $1/2$.

We can therefore write $$P(B_{n+1} > A_n) = 1 \times P(B_n > A_n) + \frac{1}{2} P(B_n = A_n).$$ We also have the identity $$P(A_n = B_n) + P(A_n > B_n) + P(A_n < B_n) = 1.$$ Let $P(A_n = B_n) = x$ and $P(A_n > B_n) = p$. By symmetry, we also have $P(B_n > A_n) = p$. Therefore, $$2p + x = 1,$$ which implies $$p = \frac{1 - x}{2}.$$ Using this expression, we have $$P(B_{n+1} > A_n) = \frac{1 - x}{2} + \frac{1}{2} x,$$ which simplifies to the surprising result $$P(B_{n+1} > A_n) = \frac{1}{2}.$$
